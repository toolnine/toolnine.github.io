🕵️ Steganography

Hide secret text inside an image or extract hidden text from an image (using LSB method). For educational purposes only.

Supported formats: PNG (Recommended)

⬇️ Download Encoded Image

How to Use

  1. Upload an image file (PNG recommended for best results).
  2. To hide a message: Enter text and an optional password, then click "Hide Message". A new image with the hidden message will be generated for download.
  3. To decode a message: Upload the encoded image, enter the optional password (if one was used), and click "Decode Message". The hidden text will appear in the result box.
