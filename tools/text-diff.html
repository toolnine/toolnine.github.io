How to Use

  1. Paste the original version of the text into the left box (Text A).
  2. Paste the modified version of the text into the right box (Text B).
  3. (New) Select "Line-by-Line Mode" to compare changes line by line, or uncheck for word-by-word comparison.
  4. Click "Compare Texts" to see the differences.
  5. Added lines will appear in Text B (green highlight), and removed lines will appear in Text A (red highlight).
